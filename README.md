# Deep Learning Course — HSE 2025–2026

Домашние задания и материалы курса **Introduction to Deep Learning** (ВШЭ, 2025–2026).

## 📂 Структура репозитория

| Папка | Описание |
|---|---|
| `bhw1-backprop/` | Большое ДЗ 1 — Backpropagation (MLP с нуля) |
| `bhw2-translation/` | Большое ДЗ 2 — Machine Translation (GRU + Transformer) |
| `shw-01-mlp/` | Малое ДЗ 1 — Multilayer Perceptron |
| `shw-02-cnn/` | Малое ДЗ 2 — Convolutional Neural Networks |
| `shw-03-rnn/` | Малое ДЗ 3 — Recurrent Neural Networks |
| `shw-04-ssl-bonus/` | Малое ДЗ 4 (бонус) — Self-Supervised Learning / CLIP |
| `checkpoint/` | Контрольная работа |
| `theory/` | Теоретическое ДЗ (LaTeX) |
| `seminars/` | Ноутбуки с семинаров |
| `misc/` | Вспомогательные скрипты и файлы |

- Python, PyTorch
